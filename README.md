# Football-EPL-Trends-Analysis-using-Powerbi
Interactive Football Analytics Dashboard built in Power BI showcasing team performance, league standings, goal trends, home vs away wins, clean sheets, and player statistics. Features dynamic KPIs, slicers, DAX measures, and premium sports-themed data storytelling visuals.
## Key Features
- Team ranking analysis
- Goal trends by month
- Home vs away win comparison
- Top scorer insights
- KPI cards and DAX measures
- Sports-themed premium UI

## Tools Used
- Power BI
- DAX
- Excel / CSV
- Sports dataset

## Data Source
Data collected using the football-data.org API (Premier League free tier) and transformed into team-level analytics for Power BI visualization.
